# 실버하벤 CRPG - 사건 B: 증폭기 정지 + 지하 기계 발견 (Scenario B)

**버전:** 1.1.0  
**최종 업데이트:** 2026-04-18
**사건 난이도:** Medium  
**중심 스킬:** 대장기술, 연금술, 은신

---

## 개요

사건 A와平行 또는 사건 A 후 3일, 새벽 2시. 갑자기 중앙 에테르 증폭기가 정지하고, 안개가 마을 외벽까지 밀려옵니다. 하플링 정보상의 제보로 노예 거주지 지하에서 정체불명의 흑마도 기계가 발견됩니다.

---

## 상황 설정

### 시간/장소
- **시간:** 사건 A와平行 또는 사건 A 후 3일, 새벽 2시
- **장소:** 
  - Phase 1: 실버하벤 중앙 에테르 증폭기실
  - Phase 2: 하층 지하 노예 거주지

### 배경 스토리
- 갑자기 중앙 증폭기가 정지
- 안개가 마을 외벽까지 밀려옴 (시야 30ft로 감소)
- 수복파장 발렌틴: "누군가 증폭기에 손을 댔다"
- 하플링 정보상: "지하 노예 거주지에서 기적이 죽어가고 있다"

### 플레이어 역할
- 증폭기 수리반으로 투입
- 또는: 노예 구출 대원으로 투입
- 또는: 둘 다 (시간 제한 선택)

### [Phase 1: 증폭기 수리]

#### 증폭기실 진입
```

│  [증폭기실 - 긴급 상황]                                     │

│    ┌──────────────────────────────────────────┐            │
│    │        ◆ 증폭기 (정지 상태)              │            │

│    └──────────────────────────────────────────┘            │
│                                                              │
│  [적: 흑마도 사도 2기]        [기술 장치 파편]               │
│         ↓                           ↓                       │
│  ┌───────────┐               ┌───────────┐               │
│  │ 수리 필요:│               │ 수리 필요:│               │

│  └───────────┘               └───────────┘               │

│  [출구]                            [계단 -> 지하]           │

```

#### 전투: 흑마도 사도 2기 vs 플레이어

**흑마도 사도 기본 정보:**
```
HP: 45 | AC: 13 | SPD: 30ft

특성:
- 노이즈 폭발: 범위 10ft, 2d6 + 마법 대미지, DC 13 내성
- 정신 장악: Bonus Action, 1명 1턴 통제 시도 (WIS DC 13)
```

**전투 팁:**
- 사도들은 증폭기실 中央에 위치
- 플레이어는 입구에서 수리 장치에 접근 가능
- 장치 수리 중에는 기회 공격에 노출됨

#### 수리 성공 조건

| 수리 항목 | 필요 스킬 | DC | 시간 |
|-----------|----------|-----|------|
| 전원 장치 | 대장기술 또는 연금술 | 15 | 2턴 |
| 마도핵 | 연금술 | 18 | 3턴 |

**성공:**
```
2개 장치 수리 → 증폭기 재가동 → 안개 퇴각 (地上 완료)
XP: 대장기술 +50xp, 연금술 +50xp
```

**실패:**
```
시간 5분 소요, 안개 추가 침투
HP: -10% (안개 오염)
```

#### 노예화된 수문병 전투 (선택: 중립화 or 처치)

**노예화된 수문병 기본 정보:**
```
HP: 30 | AC: 14 | SPD: 30ft

특성:
- 돌진:Charge 10ft, 1d8+2 대미지
- 방패 방어: AC +2 (盾使用時)
```

**대화 옵션 (중립화):**
```
플레이어: "너희도 제물이야?"

수문병: "... ..."

[Speech Check: CHA 12+ or INT 14+]
→ 성공: 수문병이 자리를 비켜줌
→ 실패: 전투 발생
```

#### 제단 발견

```
┌────────────────────────────────────────────────────────────┐
│  【제단 발견】                                              │
│                                                              │
│  돌로 된 제단 위에 노란 표식이 그려져 있다.                  │
│  주변에 희미한 피 냄새와 마력의 잔향이 남아있다.              │
│                                                              │
│  [힌트] "매일 밤 제물 바침" → 기계에 에너지 공급            │
│                                                              │
│  [선택지]                                                   │
│  ├─ A: 제단 파괴                                           │
│  ├─ B: 제단 조명 (증거 수집)                                │
│  └─ C: 무시하고 진행                                        │
└────────────────────────────────────────────────────────────┘
```

| 선택 | 결과 |
|------|------|
| **A: 제단 파괴** | 기계 에너지 공급 감소, 다음 전투 쉬워짐 |
| **B: 제단 조명** | 증거 확보, 심연의 사도 증거로 활용 가능 |
| **C: 무시** | 추가 정보 없음, 전투 난이도 유지 |

### [Phase 3: 보스전 - 기계 수호자]

#### 보스: 기계 수호자 (CR 5)

```
┌────────────────────────────────────────────────────────────┐
│  【기계 수호자】                                            │
│                                                              │
│  HP: 85 | AC: 16 | SPD: 25ft (방해로慢)                   │
│                                                              │
│  【특성】                                                   │
│  - 회전톱: 2d8 + 방어구 관통                                 │
│  - 자기장: 반경 10ft 내 장비 무력화 (마법 무기만有效)        │
│  - 자가 수리: HP 10/턴 회복                                 │
│                                                              │
│  【약점】                                                   │
│  - 노바 금편 → 방어구 관통                                   │
│  - 제단 파괴 시 → 자가 수리 비활성화                         │
│                                                              │
│  【 phases】                                                │
│  Phase 1 (HP 85~50): 기본 행동 패턴                         │
│  Phase 2 (HP 50~0): 자가 분노, 공격력 +50%                  │
└────────────────────────────────────────────────────────────┘
```

#### 전투 전략

| 전략 | 효과 | 필요 스킬/아이템 |
|------|------|-----------------|
| **고효율 무기 사용** | 마법 무기만 대미지 가능 | 연금단검, 마도 무기 |
| **제단 파괴 후 전투** | 자가 수리 비활성화 | - |
| **노바 금편 공격** | 방어구 관통 | 노바 금편 (사건 C) |
| **빠른 처치** | Phase 2 진입 차단 | DPS 캐릭터 필요 |

#### 전투 후 선택

**기계 파괴:**
```
기계가 산산조각 나면서 노예들의 비명이 멈춘다.

플래그: `void_machine_destroyed` = true
XP: +150 XP
보상: 기계 설계도 (히든 루트 잠금)
```

**기계 작동 정지:**
```
플레이어가 기계의 핵심을 비활성화한다.
기계는 멈추지만, 내부에서 빛이 사라지지 않는다.

플래그: `void_machine_stopped` = true
XP: +100 XP
보상: 기계 설계도 (히든 루트 가능)
힌트: "이 기계는 다시 가동될 수 있다"
```

### [Phase 4: 심연의 사도 추적]

#### 히든 출구 발견

```
[히든 출구] ← DEX 체크 (DC 15) 또는 INT 체크 (DC 12)

발견 시:
└─ 심연의 사도가 탈출로를 사용하려 함

[선택]
├─ 추적: 전투 또는 추격 체크
└─ 포기: 사도逃脱, 다음 사건에서再登場
```

**추적 성공:**
```
심연의 사도를 붙잡고 심문할 수 있다.

사도: "네가 발견했다고 해서 달라지는 건 없다.
       안개는 이미 마을을 덮었다.
       우리는 다만... 되찾는 것뿐이다."

힌트: 심연의 사도는 "다시 되찾는다"고 표현
       → 인간이 뺏은 것을 되찾는다는 의미?
       → 노예들의 원한을 증폭시키는 이유와 연결
```

## 사건 후속

사건 B 완료 후:

```
┌─────────────────────────────────────────────────────────────┐
│  사건 B: 지하 기계 (완료)                                   │
│                                                              │
│  【플래그 영향】                                            │
│  - 심연의 사도 추적 성공 → 사건 C에서 추가 정보              │

│                                                              │
│  【다음 사건】                                               │
│  ├─ 사건 C: 노바 부유 도시 잔해 (권장)                       │
│  └─ 자유 탐험                                               │
│                                                              │
│  【히든 루트】                                               │
│  - 기계 정지 + 설계도 획득 + 심연의 사도 추적 성공            │
│  → 사건 C에서 노바 망령에게 "기계를 재가동할 수 있다"提案 가능│
└─────────────────────────────────────────────────────────────┘
```
