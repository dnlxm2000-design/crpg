## 1. 스킬 개요

### 1.1 시작 스킬 배분
- 처음 시작 시 **100 수련치**를 4가지 스킬에 분배
- 한 스킬당 최대 **50**까지 올릴 수 있음
- 올리기 어렵거나 돈이 드는 스킬 2개를 50까지 올리고, 2개는 적당히 내리는 것이 좋음
- 약 **30**까지의 스킬은 NPC에게 적은 돈을 주고 배울 수 있음
- 지정한 4가지 스킬에 따라 처음 가지고 시작하는 아이템이 달라짐

### 1.2 스킬 레벨 체계
| 수치 | 타이틀(한글) | 타이틀(영문) |
|------|-------------|-------------|
| 10 | 견습생 | Neophyte |
| 20 | 초보 | Novice |
| 30 | 수습 | Apprentice |
| 40 | 기술자 | Journeyman |
| 50 | 숙련자 | Craftsman |
| 60 | 장인 | Artisan |
| 70 | 전문가 | Adept |
| 80 | 상급자 | Expert |
| 90 | 달인 | Master |
| 100 | 거장 | Grand Master (GM) |
| 110 | 명장 | Elder |
| 120 | 전설 | Legendary |

- 기본 최대 **100.0**, 확장팩 "로드 블랙쏜의 복수"에서 챔피언 몹 스크롤로 **120**까지 확장
- 모든 스킬 총합 기본 **720.0** 초과 불가 (본래 700 → 2002년 베테랑 계정 720 → 2013년 전체 계정 720)
- 스킬에는 **올리기, 잠그기, 내리기** 설정 가능. 총합이 채워지면 불필요 스킬을 내려 다른 스킬 상승

### 1.3 스킬 메커니즘
- **성공률계 스킬**과 **효과량계 스킬**로 구분
  - 예: Magery = 마법 시전 성공/실패, Evaluating Intelligence = 마법 데미지량
- NPC에게 돈으로 배움: **1골드 = 스킬 0.1** (예: 297골드 → 29.7)
- NPC 직업에 따라 가르치는 스킬이 다름 (마법사 NPC → 마법학/마법저항, 대장장이 → 장비지식/대장기술)
- **스킬핑퐁**: 스킬 수련 시 특정 스탯 상승 → 스탯 맞추기 위해 스킬 올렸다 내렸다
  - 무기 기술 → 힘+민첩, 은신/훔치기 → 민첩, 지능측정/마법/심령술 → 지능
- 타이틀은 선택적으로 부착 가능 (달인 전사도 숙련 치료사 타이틀 부착 가능)

## 2. 스킬의 종류

### 2.1 연금술 (Alchemy)
- **시작 아이템**: 3~4개 랜덤 시약, 빈병 4개, 막자와 막자사발, 로브
- **타이틀**: 연금술사(Alchemist)
- 시약을 갈아 물약 제작 (해독/회복/폭약/독약/안약)
- Renaissance: 연금술 10당 폭탄 데미지 +1 (방어력 무시)
- Third Dawn: 유리 세공 — 해변/사막 모래로 플라스크/모래시계 제작
- 식물 컨텐트: 화분 벌레 퇴치, 식물 영양제/농약

### 2.2 해부학 (Anatomy)
- **시작 아이템**: 붕대 3개, 로브
- **타이틀**: 생물학자(Biologist)
- 동물/사람/몬스터 상태(스태미너, HP 등) 확인
- **패시브**: 데미지와 회복량 증가
- 스킬 61 이상 → 붕대로 **해독** 가능
- 스킬 81 이상 → 붕대로 **부활** 가능

### 2.3 동물 지식 (Animal Lore)
- **시작 아이템**: 로브, 양치기 지팡이
- **타이틀**: 자연인(Naturalist)
- 동물 상태 상세 스펙 확인 (숫자 표시)
- 수의학 붕대질 회복량 증가
- 스킬 61 이상 → 해독, 81 이상 → 유령 동물 부활

### 2.4 동물 조련 (Animal Taming)
- **시작 아이템**: 양치기 지팡이
- **타이틀**: 조련사(Tamer)
- 야생동물 길들이기 (말/라마/용 등)
- 용 길들이기 → 던전 탱커로 유용
- 쿠 시데: 스스로 붕대질 가능

### 2.5 궁술 (Archery)
- **시작 아이템**: 화살 25개, 활
- **적용 무기**: 활, 합성궁, 화궁(Yumi), 석궁, 중석궁, 연발석궁
- **타이틀**: 궁수(Archer)
- GM vs GM 명중률 50%
- 오리지널: 이동 중 사격 가능 → 인기
- Renaissance: 이동 정지 패치, 특수공격 없음 → 천대
- Age of Shadows: 이동 사격 페널티 완화, 특수공격 추가 → 재인기
- 가고일: 궁술 불가

### 2.6 장비 지식 (Arms Lore)
- **시작 아이템**: 랜덤 견습무기 1개
- **타이틀**: 무기 장인(Weapons Master)
- Age of Shadows 이후: 제작 제품 성능 관여
- 모든 제작기술에 필요한 스킬

### 2.7 구걸 (Begging)
- **시작 아이템**: 지팡이
- **타이틀**: 거지(Beggar)
- 울온 3대 잉여스킬. NPC에게 돈/빵 구걸
- 마을 벗어나지 않고 돈 벌 수 있는 로망 스킬

### 2.8 대장기술 (Blacksmithy)
- **시작 아이템**: 반 앞치마, 랜덤 대장장이 도구, 채광 도구 2개, 철괴 50개
- **타이틀**: 대장장이(Smith)
- 금속 무기/갑옷 제작. GM → 익셉셔널(Exceptional) 장비 (이름 새김, 품질 우수)
- 마이닝 연동: 실패 아이템 녹여 재사용량 증가
- Age of Shadows: 저항력 수치가 방어력 기준, 가죽제보다 금속제 단단 → 무기 중심 제작

### 2.9 활/화살제작 (Bowcraft/Fletch)
- **시작 아이템**: 나무판자 14개, 깃털 5개, 축대 5개
- **타이틀**: 활장이(Bowyer)
- 활과 화살 제작/수리. GM 이름 새김 가능
- NPC 판매로 초반 돈벌이 용도

### 2.10 무사도 (Bushido)
- **시작 아이템**: 부시도 스킬 책, 하카마, 카사
- **타이틀**: 무사(Samurai)
- Samurai Empire 확장팩 추가
- 방패 들 때 페널티. 양손무기 전용
- **완벽함(Perfection)**: 명예 미덕 → 공격 성공 시 포인트 누적 → 추가 공격력 + 사망 시 회복
- 분신술: (인술-30)% 확률로 물리공격 회피

### 2.11 야영 ( Camping)
- **시작 아이템**: 침낭, 불쏘시개 5개
- **타이틀**: 탐험가(Explorer)
- 야외에서 즉시 로그아웃 가능
- 스텟 핑퐁용 (힘/지능/민첩 동시 상승)

### 2.12 목공술 (Carpentry)
- **시작 아이템**: 나무판자 10개, 반 앞치마, 목공 도구
- **타이틀**: 목수(Carpenter)
- 보관함/가구 제작. Third Dawn: 석공술 추가 (화강암 가구/석상)
- Age of Shadows: 나무 종류 다양화, 나무 갑옷 추가
- Stygian Abyss: 가고일 전용 돌 갑옷/칼

### 2.13 지도제작술 (Cartography)
- **시작 아이템**: 빈 지도 4개, 육분의
- **타이틀**: 지도제작자(Cartographer)
- 보물지도 해석 필수 스킬. 높을수록 높은 레벨 보물지도 해석
- GM → 고급 마법 무기 대량 획득

### 2.14 기사도 (Chivalry)
- **시작 아이템**: 없음
- **타이틀**: 팔라딘(Paladin)
- Age of Shadows 추가. 돈(십일조) 소비, 시약 불필요
- 리콜(Sacred Journey), 근접 공격 강화 마법 다수
- 출시 후 대부분 전사가 마법학 버리고 기사도로 전향

### 2.15 요리 (Cooking)
- **시작 아이템**: 불쏘시개 2개, 날양고기, 생닭다리, 날생선, 밀가루, 물주전자
- **타이틀**: 주방장(Chef)
- 배고픔 수치 관리. 생선 스테이크/케이크 인기
- GM 쉐프 음식에 이름 새김
- 부족 페인트: 세비지 몬스터에게 선공 안 받음

### 2.16 은신 탐색 (Detecting Hidden)
- **시작 아이템**: 망토
- **타이틀**: 정찰병(Scout)
- 은신한 사람 찾기. 투명화 해제 마법/폭발 포션으로 대체 가능

### 2.17 불협화음 (Discordance)
- **시작 아이템**: 아무 악기 1종
- **타이틀**: Demoralizer
- 대상 스탯 일시 감소. 사냥 시 유용
- 초창기: 유혹 스킬 (NPC/몬스터在自己位置로 끌어옴)

### 2.18 지능 측정 (Evaluating Intelligence)
- **시작 아이템**: 없음
- **타이틀**: 학자(Scholar)
- 마법 공격 데미지 상승 → 마법사 필수

### 2.19 펜싱 (Fencing)
- **시작 아이템**: 크리스
- **적용 무기**: 단검, 크리스, 쇠스랑, 전투 포크, 단창, 창, 기병창, 쌍날 지팡이, 라자탕, 채(Sai)
- **타이틀**: 펜서(Fencer)
- 공격속도 빠름. GM vs GM 명중률 50%
- Renaissance: 아나토미 높을 시 양손무기 **마비 공격** (3초, 피해 시 풀리지 않음) → PvP 인기
- Age of Swords: 무기별 특수공격
  - 단검: 감염의 일격 (독 주입)
  - 크리스: 방어구 관통
  - 쌍날 지팡이: 이중 공격 (2회)
  - 기병창: 낙마 (적 탈것 강제 하차)

### 2.20 낚시 (Fishing)
- **시작 아이템**: 낚싯대, 넓은챙 모자
- **타이틀**: 낚시꾼(Fisherman)
- SOS병 획득용. 스킬 81 이상 → 바다뱀(시 서펜트) → SOS 병

### 2.21 집중 (Focus)
- **시작 아이템**: 로브, 단검
- **타이틀**: 금욕주의자(Stoic)
- Age of Shadows 추가. "전사의 명상" — 마나+스태미너 회복속도 증가
- 달리기 → 스태미너 소모 → 스킬 상승 (매우 쉬움)

### 2.22 법의학 (Forensic Evaluating)
- **시작 아이템**: 없음
- **타이틀**: 탐정(Detective)
- 시체 분석 (언제/어떻게/무엇으로 죽음). 지능 스탯 상승/재미용
- 상자 사용: 최후에 상자 연 사람 이름 표시

### 2.23 치료 (Healing)
- **시작 아이템**: 붕대 50개, 가위
- **타이틀**: 치료사(Healer)
- 붕대로 치료. GM → 체력 약 40 회복 (최대 HP 100 기준)
- 해부학+치료 시너지. 셀프힐 가능
- 아나토미 높으면 붕대로 해독/부활 가능
- 치료 중 공격받으면 회복량 % 감소 → 실패
- 동물 치료 불가 (수의학 필요)

### 2.24 몰이 (Herding)
- **시작 아이템**: 양치기 지팡이
- **타이틀**: 양치기(Shepherd)
- 동물 원하는 방향으로 이동. 야생동물 한정 → 쓸모없음

### 2.25 은신 (Hiding)
- **시작 아이템**: 망토
- **타이틀**: 도적(Rogue)
- 벽 등지고 은신. 반짝이는 금속제 갑옷 불가
- Age of Shadows 이전: 전사 필수 PvP 스킬
- 은신 전사: 돈되는 몹만 골라죽임

### 2.26 마법 부여 (Imbuing)
- **시작 아이템**: 없음
- **타이틀**: 장인(Artisan)
- Stygian Abyss 추가. 마법 무기 제작/기존 무구 마법 부여
- 가고일: 종족특성 마스터 장인 → 성공률 높음

### 2.27 필사 (Inscription)
- **시작 아이템**: 빈 스크롤 2개, 빈 책
- **타이틀**: 서기(Scribe)
- 마법책 주문 → 스크롤 제작. 스크롤 = 마법 스킬 +20 간주, 시약 불필요
- Renaissance: 필사 높으면 방어 마법 효과 증가 (반사 갑옷/마법 반사)
- 룬북 제작 → 돈벌이

### 2.28 아이템 감정 (Item Identification)
- **시작 아이템**: 지팡이
- **타이틀**: 상인(Merchant)
- 아이템 내구도/상태 감정. AOS부터 인챈트/내구력 표시 → 잉여

### 2.29 벌목 (Lumberjacking)
- **시작 아이템**: 벌목도끼
- **타이틀**: 나무꾼(Lumberjack)
- 나무 잘라 재료 획득. Renaissance: 도끼류 무기 데미지 상승
- **럼잭 전사**: 벌목+도끼 → 데미지 극한 (25% 추가데미지)
- 할버드/바디치 = 폴암 (벌목 효과 없음), 전쟁 도끼 = 둔기 (벌목 효과 없음)
- 곡괭이 = 도끼 취급 (Pick'AXE')

### 2.30 둔기 전투 (Mace Fighting)
- **시작 아이템**: 몽둥이
- **적용 무기**: 몽둥이, 메이스, 마울, 해머 픽, 전투 망치, 홀, 대장장이 망치, 슬러지 해머, 전쟁 도끼, 쌍절곤, 스태프류
- **타이틀**: 보병(Armsman)
- 공격 시 스태미너 큰 피해. 갑옷 내구도 감소 → 장기전 강력
- Renaissance: 아나토미 높을 시 양손무기 **분쇄 공격** (50% 추가 피해, 스태미너 피해 포함)
- Age of Swords: 무기별 특수능력

### 2.31 마법학 (Magery)
- **시작 아이템**: 랜덤 1~4서클 스크롤 3개, 시약 30개, 마법책
- **타이틀**: 마법사(Mage)
- 마법 성공률/효과 증가. 높은 서클 주문 사용 가능
- 귀환(Recall)/마크(Mark) 마법 → 이동 필수
- Age of Shadows: 기사도 신성한 여정으로 대체 가능
- 마법사 무기 옵션: (마법학-수치)만큼 무기스킬 환산

### 2.32 명상 (Meditation)
- **시작 아이템**: 없음
- **타이틀**: 금욕주의자(Stoic)
- 제자리 명상 → MP 회복. 무거운 갑옷 불가 (가죽까지 허용)
- "마법사 갑옷" 옵션 착용 시 금속제 갑옷에서도 가능

### 2.33 채광 (Mining)
- **시작 아이템**: 삽, 곡괭이
- **타이틀**: 광부(Miner)
- 광물 채굴/주괴 가공. 철괴 수요 높음 → 초반 돈벌이
- Third Dawn: 모래/화강암 채취. Mondain Legacy: 보석류 채취

### 2.34 악기연주 (Musicianship)
- **시작 아이템**: 아무 악기 1개
- **타이틀**: 음유시인(Bard)
- 도발/평온/불협화음 성공률 증가 → 시너지 필수

### 2.35 주술 (Mysticism)
- **시작 아이템**: 없음
- **타이틀**: 주술사(Mystic)
- Stygian Abyss 추가. 단체 공격 마법 다수. 가고일 불가

### 2.36 사령술 (Necromancy)
- **시작 아이템**: 네크로맨서 스킬북, 시약 50개씩
- **타이틀**: 사령술사(Necromancer)
- Age of Shadows 추가. 뱀파이어 변신 (흡혈, 불저항 약화, 마늘 마법 불가)
- 심령술 시너지: 버프 지속시간 증가, 마법 피해량 증가
- 전사: 뱀파이어폼+무기저주 유용. 마법사: 유령 변신 → 마법 데미지 일정분 마나 전환

### 2.37 인술 (Ninjitsu)
- **시작 아이템**: 없음
- **타이틀**: 닌자(Ninja)
- Samurai Empire 추가. 분신술/암습/동물 변신
- 분신술: (인술-30)% 확률로 물리공격 회피
- 구미호 변신: 체력 증가 → 즉시 회복 (독 무시 자힐)
- 뱀파이어폼 + 분신술 콤보: 체력흡수 + 회피
- 말 타고 분신술 불가

### 2.38 방패 막기 (Parrying)
- **시작 아이템**: 나무방패
- **타이틀**: 결투사(Duelist)
- 방패로 공격 막기. 데미지 흡수/감소
- Samurai Empire: 양손무기 "자루로 막는다" → 패링 가능
- 한손무기 필수 기술

### 2.39 평온 (Peacemaking)
- **시작 아이템**: 아무 악기 1개
- **타이틀**: 음유시인(Bard)
- 적대적 대상 평온화. 전투 풀림, 타겟 상실
- 바드 사냥 필수. 위급 탈출/파티 유용
- 몬스터: 일정시간 공격 정지. 유저: 전투모드 재개 → PvP 무용

### 2.40 독바르기 (Poisoning)
- **시작 아이템**: 독약 2개
- **타이틀**: 암살자(Assassin)
- 무기/음식에 독 주입. 카르마 하락
- 4레벨 독(맹독): HP 100 플레이어 10초 내 사망
- 무기 독: 내구력 하락 페널티. 11회 공격 횟수 동안 일정 확률 독 걸림
- Renaissance: 3서클 중독/5서클 포이즌 필드 보너스
- GM: 4레벨 독 → 5레벨 치명독(Lethal Poison) 변형

### 2.41 도발 (Provocation)
- **시작 아이템**: 아무 악기 1개
- **타이틀**: 음유시인(Bard)
- 타겟 둘 지정 → 서로 싸움. 사냥 시 유용 (하나 죽고 하나 빈사 → 뒷치기)

### 2.42 함정 제거 (Remove Trap)
- **시작 아이템**: 시작 스킬로 지정 불가
- **타이틀**: 도적(Rogue)
- 던전/상자 함정 제거. 보물상자 함정 제거용
- 자물쇠 따기 50 이상 필요

### 2.43 마법 저항 (Resisting Spells)
- **시작 아이템**: 없음
- **타이틀**: 마법사(Mage)
- 마법 데미지 감소. 약한 마법 무시, 센 마법 데미지 감소
- 상태변화 마법 걸릴 확률 감소 → PK 필수
- Age of Shadows: 속성 방어 생성 → 마법 위력 약화+저주 마법 스탯 저하 감소

### 2.44 훔쳐보기 (Snooping)
- **시작 아이템**: 자물쇠따개 20개
- **타이틀**: 소매치기(Pickpocket)
- 주머니 내용물 확인. 훔치기 아님. 카르마 하락
- 실패 시 "XXX가 당신의 주머니를 뒤지고 있습니다" 메시지

### 2.45 심령술 (Spirit Speak)
- **시작 아이템**: 망토
- **타이틀**: 영매(Medium)
- 유령과 대화. Age of Shadows: 네크로맨시 연계, 시체 비례 체력 회복

### 2.46 훔치기 (Stealing)
- **시작 아이템**: 자물쇠따개 20개
- **타이틀**: 소매치기(Pickpocket)
- 플레이어/NPC/몬스터 물건 훔치기. 실패 시 범죄자 인식
- 유저 사용: 훔치기 60 이상 + 도둑 길드 가입. 10스톤 한계
- 카르마 하락

### 2.47 스펠 위빙 (Spell Weaving)
- **시작 아이템**: 없음
- **타이틀**: 마법학자(Arcanist)
- Mondain Legacy 추가. 마법진 설치, 요정/악마 소환
- **단체시전**: 다수 마법학자 동시 시전 → 효과 강화+시전 가속
- 퀘스트 주문 필요 → 경험자 추천
- 네크로맨시+스펠위빙+마법학: 픽시 몸빵, 전체공격마법 → 챔피언전 솔로잉

### 2.48 은신 이동 (Stealth)
- **시작 아이템**: 시작 스킬로 지정 불가
- **타이틀**: 도적(Rogue)
- 은신 상태로 이동. 높을수록 거리 증가+성공률 증가
- 은신 30 이상 필요

### 2.49 검술 (Swordsmanship)
- **시작 아이템**: 일본도
- **적용 무기**: 장검, 브로드소드, 바이킹 장검, 일본도, 뼈 절단기, 바디치, 낫, 할버드, 초승달 칼날, 벌목도끼, 도끼, 양날도끼, 전투도끼
- **타이틀**: 검사(Swordsman)
- 선택지 넓음. 할버드(강공격력)+일본도(빠른속도)
- Renaissance: 아나토미 높을 시 양손무기 **충격 공격** (지능 절반 → 마인드 블래스트 증폭)
- Age of Swords: 무기별 특수능력

### 2.50 전술 (Tactics)
- **시작 아이템**: 일본도
- **타이틀**: 전사(Warrior)
- 무기 공격력 상승. 무기스킬 = 명중률, 전술 = 피해량
- 무기스킬 올리면 자연스럽게 오름 → 처음부터 찍을 이유 없음

### 2.51 재봉 (Tailoring)
- **시작 아이템**: 말린 천 1개 (가위로 자르면 천 50개), 재봉도구
- **타이틀**: 재봉사(Tailor)
- 천/가죽 재단 → 옷/가죽갑옷. GM → 익셉셔널
- Age of Swords: 뼈갑옷 제작. 대량주문 → 특수 천/CBD 획득

### 2.52 맛보기 (Taste Identification)
- **시작 아이템**: 아무 포션 3개
- **타이틀**: 주방장(Chef)
- 포션/음식 맛 → 포션 종류 판정, 독 확인. 취미용

### 2.53 투척술 (Throwing)
- **시작 아이템**: 없음
- **타이틀**: 투척병(Thrower)
- Stygian Abyss 추가. **가고일 전용**. 궁술 역할
- 한손 무기 → 방패 사용 가능 (방패 막기 불가, 공격 성공률 최저 고정)
- 부메랑 → 총알 소모 없음
- 사거리 6타일: 2타일 이내 = 12% 명중률 페널티(UnderThrown), 6타일 = 47% 데미지 페널티(OverThrown)
- 무페널티: 4타일 거리

### 2.54 땜질 (Tinkering)
- **시작 아이템**: 땜질 도구
- **타이틀**: 땜장이(Tinker)
- 기계장치/금속 공예품. 가구류 아이템
- Lord Blackthorn's Revenge: 파워 크리스탈+철괴 → 골렘 제작
- APB(Anti Paralyze Box): 마비 해제 트랩상자. 스킬 39.9로 전문 제작

### 2.55 추적술 (Tracking)
- **시작 아이템**: 장화, 가죽칼
- **타이틀**: 순찰자(Ranger)
- 대상 이동 방향 추적. 나무/덤불/고양이/사슴 등 노이즈 다수

### 2.56 수의학 (Veterinary)
- **시작 아이템**: 붕대 5개, 가위
- **타이틀**: 수의사(Veterinarian)
- 힐링 동물 버전. 동물 붕대 치료 → 테이머 보조

### 2.57 격투 (Wrestling)
- **시작 아이템**: 가죽장갑
- **타이틀**: 격투가(Wrestler)
- 맨손 공격. 데미지 낮음. 명중/회피 결정용 (공격스킬 차이)
- Renaissance: 아나토미 높을 시 **마비 펀치** (4초 마비), 암즈로어 높을 시 **무장 해제**
- GM vs GM 명중률 50%
