# 실버하벤 CRPG - 4대 혈통 상세 문서 (Bloodline Guide)

**버전:** 1.1.0  
**최종 업데이트:** 2026-04-18

---

## 개요

인간 종족은 출신 왕국에 따라 **4대 혈통**으로 구분됩니다. 각 혈통은 고유한 능력과 특성을 가지며, 게임 플레이 스타일에 직접적인 영향을 미칩니다.

---

## 1. 아이언브러드 (Iron-bloods)

### 1.1 배경 스토리

북부 빙벽 수호의 Bloodline. 제국 초기 오크 유배에 가장 앞장섰던 종족입니다. 에델바이스와 아이언가드 왕국의 주축 혈통으로, 강인한 체력과 단단한 방어력이 특징입니다.

**역사적 의미:** 성왕 퀘이사와 함께 삼중 방어선 중 '빙벽'의 수호자 역할을 담당했습니다. 아이언가드 왕국의 군사적 전통을 이어받아, 현재 실버하벤 성벽 수비대의 핵심 전력을 이루고 있습니다.

### 1.2 능력치 보정

```
┌────────────────────────────────────────────────────────────┐
│  STR: +2        │  가장 강한 종족 중 하나                    │
│  DEX: -1        │  갑주의 무게에 적응, 민첩牺牲              │
│  CON: +1        │  추위와 고통에 강함                        │
│  INT: +0        │  평균 수준                                │
│  WIS: +0        │  평균 수준                                │
│  CHA: -1        │  무뚝뚝함, 말재주가 없음                   │
└────────────────────────────────────────────────────────────┘
```

### 1.3 고유 특성

| 특성명 | 효과 | 사용 횟수 |
|--------|------|----------|
| **추위 저항** | Cold 데미지 50% 감소 | دائم |
| **단단한 피부** | 마법 외피 (피해 3 감소) | 1/day |
| **돌진** | Charge 거리가 5ft 증가 | - |

### 1.4 권장 클래스

| 순위 | 클래스 | 이유 |
|------|--------|------|
| ★★★★★ | Fighter | 명중/대미지 보너스, 방어 중심 |
| ★★★★★ | Barbarian | Rage + 내성 굴림 보너스 |
| ★★★★☆ | Paladin | 방어 + 신성한 힘 |
| ★★☆☆☆ | Fighter (_ARCHETYPE_) | 가능한 조합 |

### 1.5 권장 스킬

**추천 스킬:**
- 검술
- 둔기전투
- 전술
- 해부학
- 생존

**피해야 할 스킬:**
- 마법사용 (INT penalty)
- 은신 (DEX penalty)
- 소매치기

### 1.6 시뮬레이션

- 오크 **아이언스컬**의 주요 적대 대상
- 아이언가드 출신은 "형제"로 인식
- 미스트 세일러를 "안개에 찌든 약골"로 차별
- "강한 것이 곧 정義"라는 확고한 신념

### 1.7 시작 배경

```
"아이언가드 군단에서 복무 중이던 중, 갑작스러운 소집으로 
실버하벤 성벽 수비에 투입됨. 습격 당시 하층 지구를 순찰 중이었음."
→ Tutorial 시작 위치: 하층 지구
```

## 2. 에테르 가디언 (Ether-Guardians)

### 2.1 배경 스토리

마력 감응도가 비정상적으로 높은 Bloodline. 에테리아와 노바 왕국의 지식 계층으로, 과거 제국의 아카데미 '루미나스'의 주축이었습니다.

**역사적 의미:** 삼중 방어선 중 '에테르 그리드'의 관리와 감시를 담당했습니다. 마력 흐름에 대한 직관적인 이해로, 그리드의 이상 징후를 가장 먼저 감지하는 종족입니다.

### 2.2 능력치 보정

```
┌────────────────────────────────────────────────────────────┐
│  STR: -1        │  근육보다 두뇌 발달                       │
│  DEX: +0        │  평균 수준                               │
│  CON: +0        │  평균 수준                               │
│  INT: +2        │  제국의 지식을 이어받은 학자들            │
│  WIS: +1        │  에테르 흐름에 민감                       │
│  CHA: +0        │  평균 수준                               │
└────────────────────────────────────────────────────────────┘
```

### 2.3 고유 특성

| 특성명 | 효과 | 사용 횟수 |
|--------|------|----------|
| **에테르 시야** | 암흑 + 30ft, 마법 오라 감지 | دائم |
| **마법 감응 +5%** | 마법 성공률 +5% | - |
| **지식 탐색** | 역사/마법 관련 INT 체크 +2 | - |

### 2.4 권장 클래스

| 순위 | 클래스 | 이유 |
|------|--------|------|
| ★★★★★ | Wizard | School of Evocation + 대미지 |
| ★★★★★ | Warlock | Pact Magic + 정신 계열 |
| ★★★★☆ | Bard | Lore College + 지식 스킬 |
| ★★★☆☆ | Sorcerer | 자연 마법 + 감응도 활용 |
| ★★☆☆☆ | Cleric | 마법 + 지혜 활용 |

### 2.5 권장 스킬

**추천 스킬:**
- 마법사용
- 명상
- 주문저항
- 연금술
- 사역술

**피해야 할 스킬:**
- 궁술 (DEX penalty)
- 벌목 (STR penalty)
- 투척

### 2.6 시뮬레이션

- 심연의 사도의 주요 표적 (마력 흡수)
- 노바 망령과 대화 가능성 최고
- "제국의 진정한 지배층"이라는 자존심
- 다른 혈통을 "손아귀의 노예"로 취급하는 경향
- 과거의 지식을 그리드에 기록한 것이 자신의 의무

### 2.7 시작 배경

```
"루미나스 출신으로 실버하벤의 에테르 증폭기 연구자.
Tutorial 시에는 수리반으로 투입될 예정이었으나, 
아이언가드 부대 지시를 받음."
→ Tutorial 시작 위치: 증폭기실 근처
```

## 3. 솔라 워커 (Solar Walkers)

### 3.1 배경 스토리

사막의 태양 아래 길들여진 Bloodline. 솔라리스 왕국의 궁수와 정탐으로, 현재 멸망 직전의 피난민들입니다.

**역사적 의미:** 삼중 방어선 중 '태고의 숲' 외곽을 순찰하며, 사막과 숲의 경계를 감시했습니다. 뛰어난 명중률과 추적 능력으로 정찰대의 핵심이었습니다.

### 3.2 능력치 보정

```
┌────────────────────────────────────────────────────────────┐
│  STR: +1        │  혹서里的 노동으로 단련                    │
│  DEX: +1        │  모래 위의 사냥으로 민첩                  │
│  CON: +0        │  평균 수준                               │
│  INT: -1        │  교육 기회가 적었음                       │
│  WIS: +0        │  평균 수준                               │
│  CHA: +0        │  평균 수준                               │
└────────────────────────────────────────────────────────────┘
```

### 3.3 고유 특성

| 특성명 | 효과 | 사용 횟수 |
|--------|------|----------|
| **더위 저항** | Fire/Cold 데미지 50% 감소 | دائم |
| **동체 시력** | 은신 적 감지 +5, 후방 공격 +1d4 | - |
| **명중의 축복** | 원거리 공격 첫 히트 시 +2 대미지 | - |

### 3.4 권장 클래스

| 순위 | 클래스 | 이유 |
|------|--------|------|
| ★★★★★ | Ranger | Hunter + 다중 공격 |
| ★★★★★ | Rogue | Assassin +暗杀 |
| ★★★★☆ | Fighter | Champion + 원거리 특화 |
| ★★★☆☆ | Fighter (Battle Master) | inúmer + 원거리 마스터 |

### 3.5 권장 스킬

**추천 스킬:**
- 궁술
- 투척
- 추적
- 생존
- 동물학

**피해야 할 스킬:**
- 사역술 (INT penalty)
- 집중 (STR 관련)
- 명상

### 3.6 시뮬레이션

- 오크의 노예로 전락한 역사가 많음
- "과거의羞辱"을 딛고 일어서는 중
- 아이언브러드와 마찰 ("고문하던 놈들이야")
- 본 워커 부족과 공통점: "버려진 땅의 생존자"
- 사막의烈日처럼強一处烈的 성격

### 3.7 시작 배경

```
"솔라리스 왕국의 피난민. 실버하벤에서 고용된 용병.
Tutorial 시 수복파 보조금으로 참여."
→ Tutorial 시작 위치: 서성벽 외벽
```

## 4. 미스트 세일러 (Mist Sailors)

### 4.1 배경 스토리

안개와 바다의 Bloodline. 벤투스, 미스트랄, 자유항의 항해자/도적으로, 실버하벤의情報망을 장악하고 있습니다.

**역사적 의미:** 삼중 방어선 중 '안개'와 바다의 경계를 감시하며,密輸入과 침투자를 탐지하는 역할을 담당했습니다. 직관적인 탐지 능력과 교섭 기술이 뛰어납니다.

### 4.2 능력치 보정

```
┌────────────────────────────────────────────────────────────┐
│  STR: -1        │  육체보다 교섭에 의존                    │
│  DEX: +2        │  안개 속을 누비며 자란 민첩함            │
│  CON: +0        │  평균 수준                               │
│  INT: +0        │  평균 수준                               │
│  WIS: +1        │  직관과 꾺꾺의 항해                      │
│  CHA: +1        │  말재주와 교섭 능력                      │
└────────────────────────────────────────────────────────────┘
```

### 4.3 고유 특성

| 특성명 | 효과 | 사용 횟수 |
|--------|------|----------|
| **안개 탐지** | 시야 감소 구간에서 DEX 체크 +3 | - |
| **행운의 아류** | 1일 1회 d20 리롤 가능 | 1/day |
| **밀수 천재** | 계약밴드 관련 스킬 +2 | - |

### 4.4 권장 클래스

| 순위 | 클래스 | 이유 |
|------|--------|------|
| ★★★★★ | Rogue | Thief + 다수 행동 |
| ★★★★★ | Bard | Whispers College +情報収集 |
| ★★★★☆ | Ranger | Gloom Stalker + 은신 |
| ★★★☆☆ | Fighter (Champion) | DEX 기반 전투 |

### 4.5 권장 스킬

**추천 스킬:**
- 은신
- 소매치기
- 함정해제
- 엿보기
- 도발

**피해야 할 스킬:**
- 전술 (STR penalty)
- 둔기전투 (STR penalty)
- 해부학

### 4.6 시뮬레이션

- 실버하벤 암시장의 실질적 장악자
- "돈이면 다 된다" 현실주의자
- 심연의 사도와 밀접한 거래 관계 (문제 소지)
- 하플링과天然 동맹: 정보 공유 네트워크
- 안개 속에서 살아남은狡猾함

### 4.7 시작 배경

```
"자유의 항구 출신 밀수상. 실버하벤에서 추궁을 피해 숨어살다가
Tutorial 시 강제 징집됨. 정보 제공 거래로 참전."
→ Tutorial 시작 위치: 하층 지구 뒷골목
```
