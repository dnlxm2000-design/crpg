# 실버하벤 CRPG - 사건 C: 노바 부유 도시 잔해 (Scenario C)

**버전:** 1.1.0  
**최종 업데이트:** 2026-04-18
**사건 난이도:** Hard  
**중심 스킬:** 전투, 탐험, INT checks

---

## 개요

사건 B 후 2주, 정오 (稀有的 밝은 시간). 안개 속에서 떠내려온 제국의 부유 도시 '노바' 잔해가 실버하벤 동쪽에 출현합니다. 내부에는 제국 전체의 Gold 저장고가 있다는 소문이 돕니다. 실버하벤 3파발 (수복파/상인연맹/적응파)이 각각 파견대를 조직합니다.

---

## 상황 설정

### 시간/장소
- **시간:** 사건 B 후 2주, 정오
- **장소:** 실버하벤 동쪽 안개 -> 노바 잔해

### 배경 스토리
- 안개 속에서 떠내려온 제국의 부유 도시
- 노바 왕국의 마도공학 기술結晶
- 내부에 "제국 전체의 Gold" 저장고 있다는 소문
- 실버하벤 3파발이 각각 파견대 조직

### 플레이어 역할
- 파견대中选择: 수복파 (명예+명분), 상인 (Gold+보상), 적응파 (기술+장비)
- 혹은: 독립적으로 탐험 가능 (난이도 上)

### 구현 우선순위
1. **다중 경로 시스템** - 진입 경로 선택
2. **금고 선택지** - 세 갈래 결과
3. **망령 전투/대화 시스템** - 분기 처리

### 테스트 포인트
- 모든 경로가 플레이 가능한지
- 금고 선택이 올바르게 작동하는지
- 망령 대화/전투 분기가 명확한지

### 확장 가능성
- 노바 망령의 심화 스토리
- 기계 재가동 루트의 구현
- 마왕 크로노스와의 연결
