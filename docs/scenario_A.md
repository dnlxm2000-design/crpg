# 실버하벤 CRPG - 사건 A: 실버 텅 외교관의 등장 (Scenario A)

**버전:** 1.1.0  
**최종 업데이트:** 2026-04-18
**사건 난이도:** Easy  
**중심 스킬:** 외교, 대화, INT checks

---

## 개요

Tutorial 완료 후 1주일 후, 아침. 실버하벤 중앙 광장의 최고 의회청에서 '실버 텅' 부족의 외교관 '갈라크 Iron-Tongue'가 도착하여 성왕 퀘이사의 유언장을 들고 나타납니다.

---

## 상황 설정

### 시간/장소
- **시간:** Tutorial 완료 후 1주일, 아침
- **장소:** 실버하벤 중앙 광장, 최고 의회청

### 배경 스토리
- Tutorial 이후 성벽 수리가緊急 진행 중
- '실버 텅' 부족의 외교관 '갈라크 Iron-Tongue' 도착
- "성왕 퀘이사의 유언장을 가져왔다"고 주장
- 유언장 내용: 제국이 멸망하기 전, 오크에 대한 "배상 협정"이 존재

### 플레이어 역할
- Tutorial 후: "수복파 보충병" 역할 유지
- 또는: 외교 사건 선택 시 "수복파 정치군관" 출신으로 전환 가능

### [Phase 1: 외교 청문회]

#### 선택지 A: "오크 왕국이 협상을 원한다면, 먼저 병력을 후퇴시켜라"
```
갈라크: "아이언스컬은 우리의 일부일 뿐. 
         그들은 전사들이다. 후퇴는 그들의荣誉을 건드리는 것이다."
         
결과: 
- 협상 난항
- 갈라크 신뢰도 -5
- 다음 선택지로 진행
- 발렌틴: "좋은 반응이다. 물러서지 말아라."
```

#### 선택지 B: "유언장은 무슨 내용인가?"
```
갈라크: 유언장 공개

┌─────────────────────────────────────────────────────────────┐
│           【성왕 퀘이사의 유언장 - 전문】                      │
│                                                              │
│  "나는 분열된 부족을 통합하여 제국을 세웠으나,                    │
│   그 과정에서 오크와 트롤을 유배지로 추방한 것을                 │
│   후회한다.                                                  │
│                                                              │
│   그들이 우리와 함께했다면, 그리드의 붕괴를                     │
│   막을 수 있었을 것이다.                                      │
│                                                              │
```

### [Phase 2: 유언장 검증]

*선택지 B 또는 C 선택 후 진행*

#### INT 체크 (DC 12)

**INT 14+ 성공 시:**
```
플레이어: "이 유언장의 날짜가 이상합니다."

갈라크: "무슨 뜻인가?"

플레이어: "749년은 제국이 붕괴한 해입니다. 
          성왕이 그 해에 유언장을 썼다고 주장하지만,
          그리드 폭발로 중앙 정부는 이미 마비된 상태였을 것이다."

갈라크: "... 논리적으로 맞다. 하지만 이것이 우리의 주장을 부정하지는 않는다."

[히든] 갈라크는 약간 당황한 듯한 표정을 짓는다.

결과: 
- 갈라크 신뢰도 -10
- 유언장 위조 확률 증가 (플래그: `treaty_forged` = true)
- 다음 선택지가 열린다.
```

**INT < 14 실패 시:**
```
플레이어: "유언장의 내용을 믿어도 되는 건가?"

갈라크: "성왕의 직필이 담긴 유언이다. 의심할 이유가 없다."

결과:
- 추가 정보 없음
- 카이라가 대신 지적: "날짜가 영 수상하다."
```

### [Phase 3: 심연의 사도 가담]

*任意 진행, INT 12+ 필요*

```
다크엘프 밀고자: [관찰자席에서]

"저 유언장은 사실이지만, 적용 대상이 다릅니다."

갈라크: "무슨 뜻인가?"

다크엘프: "유럽에는 오크 부족이 여러 명칭으로 존재합니다.
          유언장의 '오크'와 현대의 '오크 왕국'이 같은지를
          증명해야 합니다."

[히든] 플레이어 INT 14+ → 진실 짐작 가능:
- "오크 왕국의 영역은 유배지가 아니라 본래 땅이었다"
- 심연의 사도는 논리적으로 오크 편을 드는 듯 보이지만,
  실은 협상을 지연시켜 시간을 버는 것
```

### [Phase 4: 최종 표결]

*의회의 Formal 투표를 진행*

#### [선택 1: 유언장 인정]
```
결과: 
- 오크와 평화협정 체결
- 실버하벤 내부 분열 (수복파 vs others)
- 수복파 내부 반발: 발렌틴이 분노

플래그 설정:
- `truth_treaty_accepted` = true
- `council_faction_change` = "peace_faction"
- `valentin_trust` -= 30

보상:
- 외교 경험치 +200 XP
- 갈라크 인맥 형성
- 오크 관계 +50
```

#### [선택 2: 유언장 불인정]
```
갈라크: "후회하게 될 것이다."

결과:
- 외교단 추방
- 오크군 전체 적대
- 아이런스컬 공격 강화 (다음 사건에서 등장)
- 발렌틴 만족

플래그 설정:
- `truth_treaty_accepted` = false
- `council_faction_change` = "war_faction"
- `valentin_trust` += 20

보상:
- 수복파 충성도 +20
- Tutorial 보상 외 추가 무기 획득
- 오크 관계 -30
```

#### [선택 3: 조건부 협상]
```
갈라크: "조건이라..."

[하위 선택지]

├─ "배상금은 무엇으로 지불할 것인가?"
│   ├─ Gold 지불 (500 Gold 소모)
│   ├─ 영토 양도 (특정 지역 접근 불가)
│   └─ 노예 해방 (道德적 선택, 오크 관계大幅 향상)
│
├─ "영토 인정 대신 군사동맹을 요구한다"
│   ├─ 마왕군에 대한 공동 방어
│   ├─ 정보 공유
│   └─ 오크部落의 독립 인정
│
└─ "시간을 벌어라" (INT 14+ 필요)
    └─ 현재 상태 유지, 향후 추가 협상
```

## 게임 내 영향 (플래그)

| 플래그 | 기본값 | 영향 |
|--------|--------|------|
| `truth_treaty_accepted` | false | 오크 관계 +50/-30 |
| `treaty_forged` | false | 유언장 위조 여부 |
| `silver_tongue_trust` | 50 | 갈라크 신뢰도 (0~100) |
| `council_faction_change` | "none" | 수복파/상인/적응 |
| `dark_elf_exposed` | false | 심연의 사도 힌트 공개 |
| `valentin_trust` | 70 | 발렌틴 신뢰도 |

## 보상/결과

### 성공적으로 협상 완수 시
- 외교 경험치 +200 XP

- '노바 망령'과의 대화에서 추가 선택지
