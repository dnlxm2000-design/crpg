Elf Bloodline Expansion

- Elf bloodlines have been expanded to 7 options, with region-based data fields aligned to the unified model used across all races.
- Bloodline options (example keys): elf_bloodline_silver_guard, elf_bloodline_forest_guard, elf_bloodline_moonweaver, elf_bloodline_river_guard, elf_bloodline_mountain_guard, elf_bloodline_dawnpatron, elf_bloodline_skyweaver.
- Each bloodline includes: id, name, name_en, region, region_kor, starting_location, ability_bonus, requires_kingdom, and optional description.
- Human bloodlines remain 4 options with identical field structure to ensure UI consistency and data integrity.

- This document serves as a quick reference for future expansions (e.g., multi-region Bloodlines) and QA checks.
